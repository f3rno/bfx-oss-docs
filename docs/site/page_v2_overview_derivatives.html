API Derivatives Trading

The Bitfinex APIs are designed to allow access to all the features of the Bitfinex platform and are subject to the API Terms of Service. The end goal is to allow users to create trading platforms and highly customised and advanced trading strategies. Bitfinex APIs can be used to retrieve data and perform derivatives trading*.

*The derivatives platform is provided by iFinex Financial Technologies Limited. References to Bitfinex Derivatives on this page are references to iFinex Financial Technologies Limited. Product specifications for our derivative contracts can be found here: Product Description Funding Payment Summary
