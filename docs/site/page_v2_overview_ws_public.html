Documentation

Public Channels

URL

Public channels should use the domain: wss://api-pub.bitfinex.com/

The domain: wss://api.bitfinex.com/ Should only be used for channels that require authentication.

Connecting to a channel

Use a subscription message to subscribe to a channel:

{
   "event": "subscribe",
   "channel": CHANNEL_NAME
}

Each channel documentation page also includes example subscription messages.
